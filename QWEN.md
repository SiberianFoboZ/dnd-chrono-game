# QWEN.md — Хроники отряда (dnd-chrono-game)

## Обзор проекта

Это статический HTML-сайт, представляющий собой **летопись D&D-кампании** в виде дневников шести персонажей, входящих в отряд. Каждый персонаж ведёт свой собственный дневник с уникальным стилем оформления, отражающим его характер. Главная страница `index.html` служит меню-указателем на индивидуальные дневники.

**Тип:** Не-кодовый проект (статический HTML-контент / творческий писательский проект).
**Язык:** Русский (`<html lang="ru">`).
**Хостинг:** Локальные HTML-файлы, открываются в браузере без сборки.

## Стилистические конвенции

Каждый дневник имеет **собственный уникальный стиль**, отражающий личность персонажа:

### Артур (`Artur/index.html`)
- Шрифт: `"Ink Free", "Segoe Print", "Caveat", "Bad Script"` (рукописные)
- Формат: **A5 print-ready** с правилами `@page`, `page-break-before`, `page-break-inside`
- Палитра: светлая (`#f5f0e8` фон, `#2c2c2c` текст) — реалистичная старая бумага
- Иллюстрации: класс `.diary-image` с рандомным `--rot` для поворота, обтекание текстом (`.left`, `.right`)
- Подписи под изображениями: `.image-caption` (курсив)
- Спецэффекты: `.typo` (зачёркнутые ошибки), `.strikethrough` (зачёркнутые мысли)
- Максимальная ширина страницы: `14.8cm`

### Аза (`Aza/index.html`)
- Шрифт: **`"Corinthia"`** (кастомный, из `ofont_ru_Corinthia.ttf`)
- Палитра: **тёмная готика** — фон `#0e0a0a`, бумага `#efe4d0`/`#d9c9a8`, акцент `#8b1e2b`, золото `#b89968`
- Структура: каждый «лист» — `<article class="entry">` с лёгким поворотом (`rotate(-0.3deg)`)
- Заголовки: `<h2>` с украшением `✦ ✦ ✦` снизу
- Буквица: `p:first-of-type::first-letter` — крупная, в цвет акцента
- Звёздная пыль на фоне через `body::before` (radial-gradients)
- «Фотокарточки»: класс `.photo-attachment` с модальным просмотром по клику
- Декор: `❦` (флёр-де-лис) как разделители, `//` в эпиграфах

### Эл (`El/index.html`)
- Шрифт: **`'Comforter'`** (из `ofont_ru_Comforter.ttf`)
- Палитра: очень тёмный фон `#2a1f14`, старая бумага `#f4e8d0`
- Структура: `.book-container` с эффектом объёмной старой книги (двойной box-shadow)
- Текстура: повторяющиеся линии через `repeating-linear-gradient` (имитация волокон бумаги)
- Пожелтение по краям через два `linear-gradient` в `.book-container::after`
- Размер: `max-width: 1200px` (книжный разворот)

### Заглушки (Barandur, Malbrin, Ziraela)
- Единый минималистичный шаблон с фразой «Эти страницы ещё не написаны...»
- Шрифт как у Артура, светлая палитра
- Ссылка «← Вернуться к меню» с `href="../"`

### Главное меню (`index.html`)
- Тот же рукописный шрифт, светлая палитра `#f5f0e8`
- Список `.diary-link` со статусом: `.status.active` (● доступен, зелёный) и обычный (○ в процессе, серый)
- Цитата в подвале: «Запись обрывается. Продолжение следует, если я останусь жив.»

## Соглашения по написанию дневников

1. **HTML doctype:** всегда `<!doctype html>` + `<html lang="ru">`.
2. **Заголовок страницы (`<title>`):** «Дневник {Имя}» или полное имя персонажа.
3. **Все стили — inline** в `<style>` теге (никаких внешних CSS-файлов).
4. **Структура записи:** `<article class="entry">` или `<div class="entry">` с заголовком (h2/h3) и параграфами.
5. **Параграфы:** выравнивание `justify`, `text-indent: 1.5em`.
6. **Иллюстрации:** встроенные изображения с подписями; для Азы — модальный просмотр.
7. **Изображения персонажей:** сохраняются локально в каталоге персонажа (не использовать внешние CDN).
8. **Имена файлов изображений:** на русском, с подчёркиваниями вместо пробелов (например, `01_деревня.jpeg`, `image1.jpeg`).
9. **Шрифты:** кастомные `.ttf` файлы кладутся рядом с `index.html`; объявление через `@font-face`.

## Команды для запуска

Это **чисто статический проект без сборки**. Для просмотра:

```bash
# Вариант 1: открыть в браузере напрямую
start index.html        # Windows
open index.html         # macOS

# Вариант 2: локальный HTTP-сервер (если нужны корректные пути к шрифтам/изображениям)
python -m http.server 8000
# затем открыть http://localhost:8000
```

**Тестирование:** отсутствует (проект контентный, не программный).
**Сборка:** отсутствует.
**Линтеры:** не используются.

## Заметки и ограничения

- **Внешние шрифты:** некоторые `.ttf` файлы — резервные (например, `GreatVibes-Regular.ttf` в Aza не используется в текущем стиле, активен `Corinthia`). Не удалять без проверки.
- **Файлы `hdphoto*.wdp` в El:** формат Windows HD Photo, могут быть сырыми заготовками для конвертации в `.jpeg`/`.png`. Не все они задействованы в `index.html`.
- **Git-ignored:** каталог `.qwen/` (рабочая область Qwen Code) и файлы `*.bak`.
- **Длина файлов:** дневники очень большие (Аза — 1078 строк, Артур — 1213 строк) — при редактировании использовать `offset`/`limit` в read_file и точечные правки через `edit`.
- **Языковая конвенция:** контент полностью на русском, включая имена файлов, метаданные и комментарии в CSS. Не переводить без явного запроса.
- **Связь персонажей:** Артур и Аза уже встретились в сюжете (упоминается в записях Азы). Эл, Барандур, Малбрин и Зираэлла — часть того же отряда.
